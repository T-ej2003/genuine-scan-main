# MSCQR Workflow Charts

This folder contains role-based workflow charts for the main MSCQR users.

Files:

- `manufacturer-workflow.md`
- `licensee-admin-workflow.md`
- `super-admin-workflow.md`
- `customer-verification-workflow.md`

Each chart is written in Mermaid so it can be reviewed in Markdown viewers that support Mermaid rendering.
